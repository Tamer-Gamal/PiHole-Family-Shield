#  One URL per line. Lines starting with #  are ignored.

#
#  NOTE: no domain list is ever 100% complete. New adult sites appear daily and
#  are added to these lists over time, but some will slip through until then.
#  Treat this as a strong filter, not a guarantee - pair it with device-level
#  parental controls and open conversations.
